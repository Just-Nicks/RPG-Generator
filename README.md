# ⚔️🛡️ RPG GENERATOR
This project focuses on helping RPG masters and players develop ideas for their RPGs, and it's also a way for me to learn more about JavaScript as a developer.

## 🚀 Tecnologies 
- `HTML`
- `CSS`
- `JAVASCRIPT`

## 🛠️ Features
This project can generate:
All functions have specific settings.
- NPC's / Characters
- Enimes 
- Items 
- Spells

## ✨ Creative process
This project started with an idea I had while writing my own RPG. Then I thought of making this website.

This project also helped me understand and format JavaScript code, enabling me to become a better developer.
